## 📋 Описание

**[DarkTech] Price Editor** — это профессиональный плагин для WordPress/WooCommerce, предназначенный для массового управления ценами и атрибутами товаров. Плагин предоставляет удобный полноэкранный интерфейс, позволяющий редактировать данные товаров в реальном времени без перезагрузки страниц.

Идеальное решение для магазинов с большим ассортиментом, где требуется быстрое обновление цен, остатков или статусов.

### 🚀 Ключевые возможности

*   **⚡ Inline-редактирование:** Изменяйте цены, названия, артикулы и другие параметры прямо в таблице.
*   **🖥️ Полноэкранный режим:** Максимально эффективное использование пространства экрана для работы с данными.
*   **🔎 Мощный поиск и фильтры:**
    *   Поиск по ID, названию или артикулу.
    *   Фильтрация по категориям, статусу публикации, наличию и налоговому статусу.
*   **🛠️ Управление колонками:** Настраивайте видимость колонок под свои задачи.
*   **🕘 История изменений:** По клику на нижнюю статусную строку открывается полноэкранный popup с последними 100 изменениями.
*   **📱 Адаптивность:** Корректная работа на планшетах и мобильных устройствах.
*   **🔄 Автоматическое кэширование:** Высокая скорость работы и наличие кнопки для принудительного сброса кэша.

## 📦 Установка

1.  Скачайте архив с плагином.
2.  Распакуйте содержимое в директорию `/wp-content/plugins/darktech-price-editor/` на вашем сервере.
3.  Перейдите в админ-панель WordPress, раздел **Плагины** -> **Установленные**.
4.  Найдите **[DarkTech] Price Editor** и нажмите **Активировать**.
5.  В главном меню появится новый пункт **«Редактор цен»** (иконка 💵).

## 📖 Руководство пользователя

### Запуск редактора
Для начала работы кликните по пункту **«Редактор цен»** в меню администратора. Откроется специальный полноэкранный интерфейс.

### Работа с таблицей

#### 1. Фильтрация данных
Панель фильтров в верхней части экрана позволяет отобрать нужные товары:
*   **Статус:** Все / Опубликовано / Черновик / Личное.
*   **Категория:** Выбор конкретной категории каталога.
*   **Наличие:** В наличии / Нет в наличии / Предзаказ.
*   **Налог:** Налогооблагаем / Только доставка / Нет.

#### 2. Редактирование полей
Таблица поддерживает редактирование следующих полей:
*   ✅ **Название товара**
*   ✅ **Артикул (SKU)**
*   ✅ **Категория**
*   ✅ **Обычная цена**
*   ✅ **Цена распродажи**
*   ✅ **Статус наличия**
*   ✅ **Налоговый статус** и **Класс налога**

**Как редактировать:**
1.  Наведите курсор на ячейку (редактируемые поля подсвечиваются).
2.  Кликните для перевода в режим редактирования.
3.  Введите новое значение.
4.  Нажмите `Enter` для сохранения или `Esc` для отмены.

> **Примечание:** Поля "№", "ID", "Старая цена" и "Действия" доступны только для чтения.

#### 3. Настройка колонок
Нажмите на кнопку **«Настройка колонок»** (иконка 🔲) справа вверху, чтобы включить или отключить отображение определенных столбцов.

#### 4. История изменений
Нижняя статусная строка показывает последнее действие. Кликните по ней, чтобы открыть полноэкранную историю изменений.

В popup отображаются последние 100 сохранённых действий в трёх колонках:
*   **Дата и время**
*   **Что произошло**
*   **Автор**

История хранится в таблице базы данных `{wp_prefix}darktech_price_editor_logs`.

## 💻 API и Разработчикам

Плагин взаимодействует с сервером через AJAX API WordPress. Для работы требуется capability `edit_products`.

### Основные AJAX-действия

Все запросы отправляются на `wp-admin/admin-ajax.php`.

#### 1. Получение товаров (`darktech_pe_get_products`)
Возвращает список товаров с пагинацией и учетом фильтров.
*   **Response:** JSON с массивом товаров и данными пагинации (recordsTotal, recordsFiltered).

#### 2. Получение категорий (`darktech_pe_get_categories`)
Возвращает дерево категорий WooCommerce.

#### 3. Обновление товара (`darktech_pe_update_product`)
Обновляет конкретное поле товара.
*   **POST параметры:**
    *   `product_id`: ID товара
    *   `field`: имя поля (например, `regular_price`)
    *   `value`: новое значение

#### 4. Получение истории изменений (`darktech_pe_get_change_history`)
Возвращает последние записи журнала для popup истории.

## 🔒 Безопасность

*   **Nonce Verification:** Все запросы защищены проверкой `wp_create_nonce`.
*   **Access Control:** Строгая проверка прав `current_user_can('edit_products')`.
*   **Input Sanitization:** Все входящие данные очищаются перед обработкой.
*   **Direct Access Prevention:** Файлы защищены от прямого вызова.
